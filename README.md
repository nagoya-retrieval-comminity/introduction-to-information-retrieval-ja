# 正誤表

## 1章 論理検索

### 第3刷 P14 L14

#### 誤
```
近接操作(proximity operator)
```

#### 正
```
近接演算子(proximity operator)
```

#### 補足
二つの用語がどのくらい近いかを指定する方法とあるため、クエリを拡張する演算子と訳すと良さそう


## 16章　フラットクラスタリング

### 第3刷 P321 L2

#### 誤
```
クラスタリング仮説は，本質的には，第14章（258ページ）の近接性仮説である．
```

#### 正
```
クラスタリング仮説は，本質的には，第14章（258ページ）の連続性仮説である．
```

#### 補足
第14章（258ページ）に「近接性仮説」という単語はない。
